# 校园招聘管理系统 java287

## 项目概述

校园招聘管理系统是基于Java与SpringBoot框架开发的一套在线招聘平台。系统采用前后端分离的设计模式，前端使用HTML和Thymeleaf模板技术，通过JQuery实现动态交互；后端则运用Java、SpringBoot、MyBatis技术，并使用Websocket进行实时通讯。数据存储方面，系统使用MySQL进行持久化存储，同时结合Redis提高数据读取性能。

## 技术栈

- **后端**：Java、SpringBoot、MyBatis、Websocket
- **前端**：HTML、Thymeleaf、JQuery
- **存储**：MySQL、Redis

## 功能模块

### 用户角色

系统定义了三种用户角色：求职者、招聘者、管理员。

#### 求职者模块

- **登录注册**：用户通过输入正确的信息实现账户的登录和注册。
- **前台首页**：展示轮播图，提供职位检索的入口。
- **检索职位**：支持关键词搜索，根据学历、工作经验、工作地点等信息筛选职位。
- **检索公司**：根据公司融资情况和规模进行查找。
- **编辑简历**：在线编辑简历，包括意向岗位、期望薪资、工作经历等信息。
- **个人信息**：维护用户的基本信息，如用户名、邮箱等。
- **在线沟通**：提供与招聘者实时的在线沟通功能。

#### 招聘者模块

- **登录注册**：输入正确信息进行登录和注册。
- **职位管理**：发布新职位或对现有职位进行下架处理。
- **员工管理**：关联和管理公司员工信息。
- **公司管理**：维护公司基本信息。
- **在线沟通**：与求职者进行在线交流。

#### 管理员模块

- **登录**：管理员通过输入正确的信息登录系统。
- **用户管理**：对用户信息进行增删改查，以及用户的封禁与解封操作。
- **岗位管理**：检索和删除岗位信息。
- **公司管理**：检索、编辑和删除公司信息。
- **简历管理**：检索、编辑和删除求职者的简历信息。
- **私信管理**：查询私信记录。
- **角色管理**：控制系统角色权限。
- **报表管理**：生成求职信息的数据图形化报表。
